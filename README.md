# School Dashboard System

Multi-tenant dashboard for tailoring schools with secure data isolation.

## Features
- Multi-tenant architecture
- Secure authentication
- Admin and school dashboards
- Mobile responsive
- Auto-healing database connections

## Deployment
Deployed on Railway.app with MySQL database.
